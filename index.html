HTML - язык гипертекстовой разметки

Основные теги - р, br, pre, b, strong, i , em, sup, sub, h1-h6, hr

Атрибут style подключает CSS стили.

Ссылки создаються с помощью тега а и его атрибута href

Картинки добавляються с помощью тега img и его атрибута указания пути src

Изменить размеры картинки можно с помощью атрибутов width и height

Изменить местоположение картинки можно с помощью атрибута align

Для создания таблиц используються теги table, tr, td, th

Выровнять текст по центру колонки можно с помощью тега align и атрибута center

Соединять ячейки можно с помощью атрибутов colspan и rowspan

Подпись к таблице задаеться с помощью тега caption , а заголовки с помощью тега th
